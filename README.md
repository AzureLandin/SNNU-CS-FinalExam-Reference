## 前言

初入智算学院的你们是否对未来感到迷茫，面对种种硬核的数学理论和繁复的编程课程是否手足无措？
内向的你是否羡慕社牛同学从学长学姐那儿复刻各种实验代码，担心自己在配置环境、Debug 的起跑线上就卷不过他们？
在各种群聊和论坛中搜寻电子书、往年大作业、考试重点，是否浪费了你大量的编译与算力时间？
往往上完一门课（或是调完一个网络）后，才恍然大悟这门课早该如此高效学习，但为时已晚，总想留一些过来人的踩坑记录和调参忠告给后生们。

秉承“厚德、积学、励志、敦行”的校训，结合我们追求极致的极客精神，我们希望通过这种开源协作的方式，将小范围传播的、容易失传的实验指南、推导笔记和复习资料，变成所有智算人都能获取的、公开的、完善的知识库。
我们希望能够通过多年的积累和努力，助力大家在算法、代码和科研的道路上更高效地学习，更好地完成学业！

## 参与贡献

### 正常流程

1. **Fork** 本仓库。

2. 确保 fork 仓库和本仓库 `main` 分支同步（在你的仓库中使用 **Sync fork**）。

3. 在网页端上传文件。

4. 向本仓库提交 **Pull Request**。

5. 检查 Pull Request 内容并等待管理员进行 Merge。

> 📌 **提示：** 本项目体积较大，clone 所有内容到本地比较耗时，建议直接在网页端提交 PR。

### 若出现冲突（Conflict）

- 备份本地文件，重新 Fork 然后重新提交 PR。

- 如果您熟悉 Git 操作，可以通过 `Rebase` 消除主分支冲突。

## 警告（Bug & Security）

- **请勿** 上传盗版电子书/付费电子书。

- **请勿** 上传盗版/破解版/绿色版付费软件及其安装包。

- **请勿** 上传任课教师明确表示勿外传的内部未公开讲义、项目及考题。

- 请大家自觉遵守开源规则，**请勿** 将本仓库资料上传至其他付费解锁/下载平台。

> 💡 如果您认为本仓库的一些文件侵犯了您的权益，请提交 Issue 或直接联系我们。我们将会从仓库中彻底清除这些文件。
